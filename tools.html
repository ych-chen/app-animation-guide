重点工具说明

After Effects + Bodymovin (Lottie)

行业标准动效软件。通过 Bodymovin（现名 aejsx）插件导出 Lottie JSON。

⚠️ Lottie 导出限制：仅支持形状图层、纯色图层、图片图层和文字图层基本属性。遮罩/混合模式部分支持。特效（模糊、发光）和第三方插件效果不导出。

Rive ⭐ 推荐关注

新一代交互式动画工具，定位为"Lottie 的进化版"。支持状态机、事件系统和实时交互，运行时 < 5MB。支持 Unity/Unreal/Flutter/RN。

2.3 Lottie 工作流详解

工作原理

After Effects→ Bodymovin 插件→ Lottie JSON→ iOS / Android / Web / RN
  • Lottie JSON：矢量动画描述格式，体积仅 GIF/APNG 的 1/10~1/100
  • dotLottie：新一代格式，JSON + 图片打包为单文件，压缩更小
  • 运行时：各平台原生渲染器，实时矢量渲染，缩放不失真

React Native 接入 Lottie

import LottieView from 'lottie-react-native';

// 基础用法
<LottieView
  source={require('./assets/loading.json')}
  autoPlay
  loop
  style={{ width: 200, height: 200 }}
/>

// 控制播放
const animRef = useRef(null);
animRef.current?.play(0, 60);  // 播放帧范围
animRef.current?.pause();

2.4 资源管理最佳实践

推荐目录结构：

assets/
  animations/
    ├── lottie/          # Lottie JSON 文件
    ├── rive/            # Rive .riv 文件
    └── sequences/       # 序列帧（Git LFS 管理）

命名规范：{场景}-{组件}-{状态/动作}.json（如 home-loading-spinner.json）
